# Digital-Payment-Risk-Fraud-Analytics-using-SQL

## Overview
This project demonstrates end-to-end SQL analytics using Microsoft SQL Server 2022 on a digital payment dataset containing **20,000 transactions**, **2,000 users**, **400 merchants**, and **763 fraud cases**.

## Objectives
- Design a relational database
- Validate imported data
- Perform exploratory data analysis
- Analyze customer and transaction behaviour
- Detect fraud patterns
- Generate business insights and recommendations

## Tech Stack
- Microsoft SQL Server 2022 Express
- SQL Server Management Studio (SSMS)
- T-SQL

## Dataset
- Users: 2,000
- Merchants: 400
- Transactions: 20,000
- Fraud Labels: 763 fraudulent transactions
